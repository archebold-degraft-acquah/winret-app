getForMe: User Guide

Overview

getForMe is a lightweight Windows application designed to enhance productivity by providing quick access to word definitions and essential applications using simple keyboard shortcuts. With getForMe, you can:

  • Instantly retrieve definitions for words by selecting them and copying them to the clipboard.
  • Quickly open commonly used applications like Microsoft Word, Excel, PowerPoint, Notepad, and WordPad with keyboard commands.

This is the first major release of the software (version 1.0). Download it at https://github.com/archebold-degraft-acquah/get-for-me/getforme1.0.exe

Features

1. Get Definitions of Words

To get the definition of a word:

  1. Select the word in any text application or browser.
  2. Copy the word to your clipboard (usually by pressing Ctrl + C).

  3. Press and hold Control + Alt and then quickly press 1 twice on your keyboard like the below image: The app will automatically fetch the definitions for the word and you'll hear the definitions announced in your ears!

    2. Open Applications with Keyboard Commands

    Simplify your workflow by using the following keyboard commands to open applications:

  4. Microsoft Word: Double Press Control + Alt + 2, like this: A close-up image of a person's hands on a keyboard. The left hand is holding down the 'Control' (Ctrl) and 'Alt' keys, while the right hand is in the motion of pressing the '2' key twice quickly.

  5. Microsoft Excel: Double Press Control + Alt + 3
  6. Microsoft PowerPoint: Double Press Control + Alt + 4
  7. Notepad: Double Press Control + Alt + 5
  8. WordPad: Double Press Control + Alt + 6

Please note that the key presses are the same as shown in the above images.

So, to open microsoft word for example, whiles you are holding down the control and the alt keys, press 2 twice quickly; then word will open. Same applies to the dictionary.

3. Compatibility

  • Operating System: Windows 10 and 11 only
  • Architecture: Supports both 32-bit and 64-bit versions of Windows

Installation

To install getForMe, follow these steps:

  1. Download the app from the following link: https://github.com/archebold-degraft-acquah/get-for-me/getforme1.0.exe
  2. Run the downloaded .exe file and follow the on-screen instructions to complete the installation.

You can choose to start the app automatically after installation. Once the app is installed, depending on the settings you choose for the app; whether you want to lunch the app after installation or not; if you choose to start the app after installation, it will lunch after pressing the finish button.

To stop the app, hold down the control and the alt keys and press c twice. In the same way, to start the app when it is not running, hold down the control and the alt keys then press s twice.

Why Use getForMe?

  • Enhanced Productivity: Perform common tasks quickly with keyboard shortcuts, minimizing disruptions in your workflow.
  • Ease of Use: Simple, intuitive keyboard commands allow you to use the app with minimal learning curve.
  • Convenient Access to Definitions: Quickly look up word definitions without needing to leave your current application or browser.
  • Streamlined Application Launching: Open essential applications without navigating through menus or desktop.
